# Autoresearch Showcase — 自主研究展示

[![Live Demo](https://img.shields.io/badge/Live-autoresearch--showcase.vercel.app-22c55e)](https://autoresearch-showcase.vercel.app)

A bilingual (Chinese–English) visual explainer of Karpathy's autoresearch methodology — autonomous AI agents that run experiments, measure results, and improve themselves without human intervention.

## Sections

- **The Core Loop** — 6-step flow: Baseline → Analyze → Mutate → Run → Keep/Discard → Repeat
- **Two Applications** — ML training optimization (Karpathy) vs Claude Code skill optimization, side by side
- **Live Simulation** — Animated terminal showing a real experiment run (70% → 95% in 3 experiments)
- **Score Progression** — Canvas chart with color-coded keep/discard markers
- **Key Principles** — Binary evals, one variable at a time, baseline first, revert ruthlessly, log everything, run autonomously
- **Credits** — Links to [karpathy/autoresearch](https://github.com/karpathy/autoresearch) and [autoresearch-skill](https://github.com/olelehmann100kMRR/autoresearch-skill)

## Design

Dark terminal-inspired aesthetic with green/red keep/discard color coding, ascending experiment dots hero canvas, Geist Mono + Manrope typography, animated terminal demo.

## Tech

Single-page HTML/CSS/JS — no build step, no dependencies.
